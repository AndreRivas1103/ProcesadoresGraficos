# Procesadores Gráficos

Sitio web sobre procesadores gráficos (GPUs): qué son, aplicaciones y empresas líderes.

## Contenido

- **Página de inicio** (`index.html`): enlaces a la presentación en Canva y al sitio web.
- **Sitio principal** (`procesadores.html`): secciones ¿Qué son?, Áreas de aplicación, Empresas (Nvidia, AMD, Intel) y Casos reales.

## Tecnologías

- HTML5
- CSS3 (variables, flexbox, responsive)
- JavaScript (Bootstrap 5, navbar, scroll, modales)


## Cómo abrirlo

Abrir `index.html` en el navegador o servir la carpeta con un servidor local (por ejemplo `python -m http.server` o la extensión Live Server).
